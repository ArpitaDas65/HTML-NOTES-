NETWORKING OVERVIEW

INTRODUCTION

Networking is the foundation of the internet and web
development. It allows computers and devices to communicate
with each other, exchange data, and share resources.

1. Basic Network Components

  • DNS (Domain Name System): Translates
    domain names (like www.google.com) into IP
    addresses.

  • IP Address: Unique identifier for a device on a network (IPv4 and IPv6).

  • Address: Hardware address of a device's network interface.

  • Server: A machine that provides resources or services over a network.

  • Root Server: The top-level DNS server that manages top-level domains (TLDs).

  • TLD Server (Top-Level Domain Server): Manages domains like .com, .org, .in.

  • Authoritative Server: DNS server that holds the official records for a domain.

    • 2.Network Ports

    • Each service uses a port number to communicate.

    • Example: HTTP → port 80, HTTPS → port 443

      • 3.Networking Devices


      • Switch: Connects multiple devices in a LAN and forwards data to the correct device.
        • Router: Connects multiple networks and directs data packets between them.

      Summary

      Networking basics are essential for understanding how
      the internet and websites work, which is important before
      diving into HTML and web development.
