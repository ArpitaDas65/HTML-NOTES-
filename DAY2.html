Basic HTML Tags

Today we will learn about some basic HTML tags.

Headings

There are 6 levels of headings: <h1>Biggest heading
<h2> Second heading
<h3> Third heading
<h4> Fourth heading
<h5> Fifth heading
<h6> Smallest heading

paragraph

The paragraph tag is used for writing text:
<p>This is a paragraph </p>

Line Break & Horizontal Line

<br>is used for a line break <hr>is used for a horizontal line

Hyperlinks

Example of a link: <a href="https://www.google.com">Google

Images

Example of image tag:
<img src="image.jpg" alt="Description">

Summary

These are the main tags you need to start writing HTML pages. More advanced stuff will come in the next days
