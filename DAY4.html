Div, Classes ,IDs etc

So today we learned about divs, classes,
and IDs. They are like boxes to put
stuff in. Also header and footer.

Div

Div is just a container. You put things inside it. Example: <div>

Some text inside div

<div>

Class

Class is for giving same name to many elements.
CSS comes later but you need it for group stuff.

Example:

<p class="myclass">Happy Durga Puja</p>

ID

ID is unique, one per page. Use it to identify one element.

<div id="myid">This is unique div</div>

Header and Footer

Header is top part, footer is bottom.
Body is main part.

<header>This is header </header> <footer>This is footer </footer>

Summary

Divs, classes, IDs, header, footer.
That's basically it for today. Simple stuff.
