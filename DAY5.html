Forms and Attribute

So today we are going to learn about forms.
Forms are used to take input from users like
name, email, password, etc.

Form Tag

Forms start with form and end with form. Example:

input type="text" placeholder="Enter name"
input type="submit" value="Submit"
/form

    Input Types
  • text - normal text input
  • password - hides what we type
  • email - for email input
  • submit - button to send form
  • radio, checkbox - choose options
  • textarea - multi-line text
  • select - dropdown list

Form Attributes

  • action - where the form submits
  • method - GET or POST
  • placeholder - hint text in input
  • required - makes input mandatory

Summary

Forms help users submit data.
Inputs, placeholders, and attributes
control what the user enters.
