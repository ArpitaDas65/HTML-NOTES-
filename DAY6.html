Media Tag

Today we will learn about how we can
add videos, audios in an webpage.
and the uses of Iframe tag. Also going
to learn that how we can link multiple
pages together to make a multipage website.


Video Tag

Video tag is used to show videos
on web pages. Example:

video src="video.mp4" controls
/video

Audio Tag

Audio tag is used to play audio files.
Example:

audio src="audio.mp3" controls
/audio

Iframe Tag

IFrame is used to embed another
webpage inside your page. Example:

Iframe src="https://www.google.com" width="300" height="200"
/iframe

Summary

Media tags help to show videos, audios,
and embed pages. Linking multiple
pages makes our website navigable.
